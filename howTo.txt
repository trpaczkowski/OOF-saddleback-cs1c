how to create folder on desktop
  must download GIT from browser
  open command prompt
  cd Desktop
  git clone (link from github)
  cd OOF-saddleback-cs1c

*when opening the cmd after youve already cloned, use "git pull" to gain the
new file

git status
  tells you the status of the files and project
  
to use our respective files
  git checkout (your Wrk file)
		it is your first name all lowercase e.g. taylorWrk

how to add to a file
  git status - everything changed will be red
  git add *
  git status - everything changed will be green now
  git commit -m "some message" (must insert message or comp will freak out)
  git push
